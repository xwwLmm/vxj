# vxj

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run dev
```

### Compiles and minifies for production
```
yarn run build
```

### Compiles components to lib for publish

```
yarn lib
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
